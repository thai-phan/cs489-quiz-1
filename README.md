# Contacts CLI

A small Java command-line application for managing contacts.

## Features
- Create, update, delete, and list contacts
- Search contacts by name, company, title, phone number, or email address
- Add phone numbers and email addresses
- Merge duplicate contacts

## Run

```bash
./gradlew test
./gradlew run
```

## Notes
- Contacts are stored in memory for the lifetime of the app.
- Contact names may be partially known, but phone numbers and email addresses must be valid.
